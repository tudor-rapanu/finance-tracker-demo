# Getting Started

This guide walks you through setting up the FinanceTracker environment on your local machine for development and testing.

## Prerequisites

Before you begin, ensure you have the following installed:

- [.NET 10.0 SDK](https://dotnet.microsoft.com/download)
- An IDE (Visual Studio 2022, JetBrains Rider, or VS Code with the C# Dev Kit)
- Access to the database is provided via Azure DB

## Local Setup

**1. Clone the repository**

```bash
git clone [https://github.com/tudor-rapanu/finance-tracker-demo.git](https://github.com/tudor-rapanu/finance-tracker-demo.git)
cd FinanceTracker
```
